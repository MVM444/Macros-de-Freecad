# 📁 Metodología para Programar con IA (Proyectos Activos)

## 🎯 Objetivo
Estructurar el trabajo con IA para:
- Evitar prompts largos y repetitivos
- Mejorar precisión del código
- Reutilizar lógica en proyectos

---

## 📂 Estructura del Proyecto

/proyecto_x  
├── contexto.md  
├── objetivo.md  
├── pseudocodigo.md  
├── tareas.md  
├── pruebas.md  
├── instrucciones_ia.md  

---

## 📄 Descripción de Archivos

### contexto.md
Define el entorno y problema:
- Tecnologías (FreeCAD, Python, etc.)
- Situación actual
- Limitaciones

---

### objetivo.md
Describe qué se quiere lograr:
- Resultado esperado
- Alcance claro

---

### pseudocodigo.md
Define la lógica paso a paso:
- Algoritmo
- Flujo de trabajo
- Validaciones

👉 Es el archivo más importante

---

### tareas.md
Lista de trabajo:
- Pendientes
- Avances
- Control del desarrollo

---

### pruebas.md
Casos de prueba:
- Escenarios válidos
- Errores esperados
- Validaciones

---

### instrucciones_ia.md
Reglas para la IA:
- Cómo debe responder
- Restricciones técnicas
- Estilo de código

---

## 🔄 Flujo de Trabajo

1. Definir problema → contexto.md  
2. Definir objetivo → objetivo.md  
3. Crear lógica → pseudocodigo.md  
4. Solicitar código a IA  
5. Ejecutar pruebas → pruebas.md  
6. Ajustar tareas → tareas.md  

---

## ⚙️ Uso con IA

Ejemplo de instrucción:

"Usar contexto.md + objetivo.md + pseudocodigo.md  
Generar código en Python para FreeCAD"

---

## ✅ Beneficios

- Código más preciso  
- Menos errores  
- Reutilización de lógica  
- Mejor control del proyecto  
- Escalable a múltiples proyectos  

---

## 🧠 Concepto Clave

> No trabajar con prompts sueltos  
> Trabajar con estructura y lógica documentada

---

## 🚀 Aplicación

Especialmente útil para:
- Macros en FreeCAD  
- Automatización en Python  
- Especificaciones técnicas  
- Desarrollo asistido por IA
