# HVAC Evaporator STEP Placeholders

This folder contains placeholder STEP models for early HVAC insertion tests.

Units:
- millimeters (mm)

Origin convention:
- centered in XY
- base at Z = 0

Files:
- `placeholder_cube_500.step`: 500 x 500 x 500
- `placeholder_square_plate_1000x1000x30.step`: 1000 x 1000 x 30
- `placeholder_wall_prism_900x260x220.step`: 900 x 260 x 220
- `placeholder_duct_prism_1200x360x320.step`: 1200 x 360 x 320
- `Pared_9000.step`: 760 x 230 x 220
- `Pared_12000.step`: 900 x 260 x 220
- `Pared_18000.step`: 1040 x 280 x 240
- `Cassette_24000.step`: 600 x 600 x 320
- `Cassette_36000.step`: 840 x 840 x 350
- `Ducto_36000.step`: 1200 x 360 x 320
- `Ducto_60000.step`: 1600 x 450 x 380

Generator script:
- `_generate_step_placeholders.py`
- `_generate_step_catalog.py`
