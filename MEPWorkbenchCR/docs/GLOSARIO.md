# Glosario del Proyecto

## Recinto

Espacio arquitectonico que actua como unidad base de calculo de carga termica.

## Evaporadora

Equipo interior de HVAC que entrega capacidad de enfriamiento a uno o varios recintos.

## Condensadora

Equipo exterior o tecnico que respalda una o varias evaporadoras y valida cobertura global del sistema.

## Puerto

Punto de conexion tecnico de un objeto HVAC para enlazar servicios y rutas.

## Ruta

Trayecto funcional entre puertos que representa un servicio HVAC especifico.

## Cobertura

Relacion porcentual entre capacidad disponible y carga requerida de un recinto o sistema.

## Factor climatico

Indice de referencia para estimar carga termica por area, expresado principalmente en BTU/h por m2.

## Calculo rapido

Metodo preliminar de carga termica para dimensionamiento inicial y decisiones tempranas.

## Calculo preciso

Metodo de carga termica con mas variables y ajustes para aproximacion tecnica mas fina.

## Simbolo 2D

Representacion plana para lectura tecnica en planta de un objeto del sistema.

## Objeto 3D

Entidad geometrica en modelo espacial que representa un equipo o componente real.

## Sistema HVAC

Conjunto de recintos, equipos, puertos, rutas y validaciones que resuelve climatizacion del proyecto.
